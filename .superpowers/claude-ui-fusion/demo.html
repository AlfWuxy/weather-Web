把 Claude 的暖橙视觉系统，融合进当前已上线的照护功能页。

这页不是正式改动，只是给你看方向。它保留了之前生成的“今日行动、照护工作台、AI 疾病预测、新增监测对象、备选联系人”等小插图，同时吸收 Claude 版更活力、更干净的橙色系统。

我的初步判断
  • Claude 的配色和卡片质感更好，适合继续吸收。
  • Claude 的模板变量和当前后端不完全一致，不能直接覆盖。
  • 当前版本的插图资产和响应式保护更完整，应该保留。
CLAUDE 优化点

更统一的暖橙系统

背景从偏白变成奶油暖底，按钮、表单、卡片、导航都使用同一组双橙 token，视觉比当前版本更轻快。

和当前版本不同

结构更像 2 栏工作台

Claude 把“今日行动”和“7 日预测”放左边，把快捷入口和家人概览放右边。当前版本更像自适应卡片流。

需要谨慎

老人端不能太小太淡

Claude 的精致感主要来自小字号、浅边框和轻阴影。年轻人喜欢，老人端需要保留更大字号和更明确按钮。

方案 A+C · 饱和橙主视觉双栏融合

保留 A 的 2 栏工作台结构，把 C 的橙色品牌感放到顶部主视觉。这里先把 32°C 的橙色调淡，真实页面会按温度线性调整饱和度。
