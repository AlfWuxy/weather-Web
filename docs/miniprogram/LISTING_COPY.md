# 微信小程序上架文案与审核路径

## 基本资料

- 小程序名称：宜老天气通
- 建议版本号：`1.0.0`
- 品牌头像：`static/brand/yilao-avatar.png`，512 × 512 PNG
- 一句话简介：面向都昌县老人、家属与社区的天气预警和行动提醒工具。
- 详细介绍：查看都昌县实时天气、七日预报、官方预警、社区脆弱性和避暑资源，并把风险转换为补水、降温、联系家人等可执行行动。登录后可管理家庭成员、健康筛查、日记、用药记录和行动确认。

正式提交前应在微信公众平台后台按最新可选范围确认服务类目。类目必须与“天气信息、生活服务、家庭照护记录”的实际功能一致，避免使用医疗诊断、互联网医院或在线问诊表述。

## 版本说明

```text
首个微信小程序正式版。提供都昌县天气、七日预报、官方预警、今日行动、社区脆弱性、避暑资源和热暴露地图；支持微信登录后的家庭照护、健康筛查、健康日记、用药记录与行动闭环。公共天气每 30 分钟更新一次，普通页面不会直接请求第三方天气服务。用药和求助仅保存记录，不自动发送通知。
```

## 审核人员体验路径

1. 打开小程序，无需登录即可进入首页。
2. 依次打开“预报”“官方预警”“今日行动”“社区”“避暑资源”“热暴露 GIS”。
3. 在“我的”中查看隐私说明、数据来源、更新时间和医疗边界。
4. 如需审核登录功能，点击“微信登录”，主动勾选隐私同意后完成登录。
5. 登录后在“照护”中添加一位测试家人，完成健康筛查、日记、用药记录和今日行动。
6. 在“账号与隐私”中验证退出和申请注销入口。

审核备注建议写明：公共功能不要求审核员提供个人健康信息；测试健康记录可以使用虚构内容，提交后可以立即注销测试账号。

## 截图清单

开发者工具和真机验收时各留一套不含真实个人信息的截图：

1. 首页天气、风险与更新时间。
2. 七日预报或官方预警。
3. 社区与避暑资源。
4. 家庭照护首页，使用虚构称呼。
5. 隐私说明与账号注销入口。

截图中不得出现 AppSecret、天气 Key、Bearer Token、OpenID、真实老人姓名、慢病、日记、用药或服务器内部信息。
