# Web 与微信小程序功能对应表

| Web 模块 | 小程序实现 | 发布边界 |
| --- | --- | --- |
| 首页天气与热风险 | 首页原生卡片、快照时间和 stale 状态 | 公开 |
| 七日天气与健康提示 | 七日预报页、逐日行动标签 | 公开 |
| 官方预警 | 预警页、无预警和不可用状态 | 公开 |
| 公开风险与行动建议 | 今日行动页、确认状态 | 公开查看，保存需登录 |
| 避暑资源 | 列表、筛选、原生地图打开 | 公开 |
| 社区风险 | 聚合社区卡片与地图 | 公开聚合数据 |
| 1 km 热暴露 GIS | 延迟加载 Canvas、图层与方法说明 | 公开聚合数据 |
| 透明度与信任网络 | 原生说明页和指标解释组件 | 公开 |
| 登录注册 | `wx.login`，兼容已有 Web Token | 用户主动同意后 |
| 家庭成员与照护配对 | 老人列表、新增、编辑、预警、话术 | 登录、owner scope |
| 健康风险评估 | 五项筛查、结果与方法边界 | 登录，不作诊断 |
| 健康日记 | 列表与新增 | 登录、owner scope |
| 用药提醒 | 列表、新增、停用或删除 | 登录、owner scope |
| 行动确认、求助、复盘 | 原生行动闭环 | 登录或安全行动入口 |
| 推送设置 | 可选设置，默认关闭 | 需单独同意 |
| 用户、社区和避暑点后台 CRUD | 保留 Web 管理后台 | 强权限 |
| 原始病历 | 保留 Web 管理后台 | 禁止进入普通小程序 |
| 深度研究分析与导出 | Web 完整版，小程序展示公开摘要 | 强权限或研究使用 |

普通用户相关功能在小程序内原生实现。管理、原始医疗数据与研究导出保留 Web，是权限与隐私隔离方案。
