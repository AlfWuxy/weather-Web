# 微信小程序上架检查清单

## 账号与工程

- [ ] 正式小程序账号已完成主体认证。
- [ ] `project.config.json` 已替换为正式 AppID。
- [ ] 服务器已配置同一 AppID 和 AppSecret。
- [ ] `https://yilaoweather.org` 已加入 request 合法域名。
- [ ] 合法域名使用有效 HTTPS 证书、443 端口，无重定向到 HTTP。
- [ ] 生产包没有 `project.private.config.json`、密钥、测试账号密码或本地地址。
- [ ] 小程序名称、图标、简介、服务类目和版本号已在账号后台填写。

## 隐私与健康信息

- [ ] 首次启用账号或健康功能前展示隐私说明并取得主动同意。
- [ ] 平台隐私保护指引中的数据类型与实际代码一致。
- [ ] 已说明微信登录标识只保存加 pepper 的哈希。
- [ ] 已说明年龄、性别、慢病类别、日记、用药提醒和行动状态的用途。
- [ ] WxPusher 默认关闭；若启用，单独披露第三方传输范围并取得同意。
- [ ] 设置页提供退出、解绑或账号删除入口。
- [ ] 管理员原始病历、姓名、医生、诊断、主诉和病史不会进入普通小程序 API。
- [ ] 医疗边界文案明确“行动筛查与通用提醒，不提供诊断、处方或治疗建议”。

## 功能

- [ ] 未登录也能看到都昌天气状态、预报、预警、行动建议、社区与避暑信息。
- [ ] 微信登录成功，已有 Web Token 兼容绑定成功。
- [ ] 老人新增、编辑、列表和 owner 权限正确。
- [ ] 健康评估、日记、用药提醒和行动闭环可完成。
- [ ] 空数据、旧数据、服务不可用和 401 状态均有清晰页面。
- [ ] GIS 延迟加载，不阻塞首屏；加载失败仍能查看方法和统计摘要。
- [ ] 分享卡片不包含老人姓名、慢病、用药或日记内容。

## 额度与网络

- [ ] 服务端 timer 为 30 分钟。
- [ ] canonical location 为 `116.20,29.27`，默认预热列表只有都昌县。
- [ ] 月度预算上限和 fail-closed 已启用。
- [ ] 小程序客户端缓存为 30 分钟，并发请求已合并。
- [ ] 普通 bootstrap 请求不会调用 QWeather。
- [ ] 多轮自动化、模拟器和视觉测试使用 fixture，真实 QWeather 调用数为 0。
- [ ] 最终受控真实联调已记录调用前后预算差值。

## 质量与审核

- [ ] Python 全量测试通过。

- [ ] 微信开发者工具无编译错误、页面路径错误和域名错误。
- [ ] Android 与 iOS 真机完成首页、弱网、登录、照护、地图和大字号检查。
- [ ] 审核账号或审核路径可用，说明中写清无需登录即可体验的公共功能。
- [ ] 隐私说明、用户协议、数据来源和更新时间入口可达。

## 发布后

- [ ] 观察 401、5xx、bootstrap 延迟和 snapshot age。
