# 宜老天气通用户协议

版本与生效日期：2026-07-18

## 1. 服务内容

宜老天气通提供都昌县级天气、七天预报、气象预警、通用防护行动、社区公开背景、避暑资源和热暴露地图。用户登录并主动同意隐私说明后，可自愿保存家人档案、日常健康观察、用药记录和照护行动。

## 2. 账号与授权

用户应使用本人有权管理的微信账号和家庭资料。涉及家人信息时，应事先取得对方同意。备用登录码和登录状态不得转发给他人。

## 3. 数据来源与时效

天气、预警和地图结果以页面标注的数据来源、发布时间、更新时间和适用范围为准。公共数据以 30 分钟作为新鲜度期限；网络异常时可能继续展示明确标记的上一份快照。重要决定前应核对当地官方最新信息。

## 4. 健康与紧急情况

健康筛查、日记、用药和行动清单只作家庭记录与日常行动参考，不提供诊断、处方、在线问诊或治疗建议。出现紧急症状时，应立即联系当地急救或线下医疗机构。

## 5. 可选第三方服务

WxPusher 默认关闭。只有用户在设置页明确勾选第三方传输说明并开启后，系统才会发送 WxPusher UID、都昌县级预警标题与正文及一次性点击链接。不会发送家人姓名、健康日记、用药记录或家庭地址。

## 6. 使用规则

用户不得提交违法内容、冒用身份、绕过权限、攻击服务或滥用接口。服务因维护、安全风险或不可抗力需要调整时，应尽量提供清晰状态说明。

## 7. 退出、变更与联系

重要规则变化会通过版本更新提示。用户可退出登录或在“账号与隐私”申请注销，也可通过微信小程序右上角菜单中的“反馈与投诉”提交问题。运营者与联系渠道以微信平台隐私保护指引展示的认证信息为准。

正式提交前，将微信认证的个人主体姓名和专用联系邮箱同步填写到微信公众平台隐私保护指引与审核材料中。仓库不保存个人证件、AppSecret 或其他认证隐私材料。
